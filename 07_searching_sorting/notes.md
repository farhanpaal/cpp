C++ provides both manual implementations and optimized built-in functions for searching and sorting data.

**Searching**: Searching is the process of locating a specific element within a collection
- __Linear Search__: Check each element one by one, Works on unsorted data, Simple but slow
- __Binary Search__: works only on sorted data, Divide the array into halves repeatedly,Much faster than linear search
  It's known as binary, because it divides sorted list into two sublists

**Sorting**: Sorting involves arranging elements in a specific order, typically ascending or descending
- __Bubble Sort__: Compare adjacent elements and swap, Largest element “bubbles” to end, very slow
- __Selection Sort__: Find smallest element and place it at beginning
- __Insertion sort__: Insert element at correct position like playing cards, Good for small datasets
- __Merge Sort__: Divide array → sort → merge. Efficient and stable
- __Quick Sort__: Pick pivot → partition → sort.Very fast in practice
- __Heap Sort__: uses heap data structure
